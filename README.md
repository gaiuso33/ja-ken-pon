# ja-ken-pon
a javascript rock-paper-scissors game project 
Rock Paper Scissors (JavaScript)

A simple Rock–Paper–Scissors game built with vanilla JavaScript as a learning exercise.

##Description

This project is a console-based Rock–Paper–Scissors game where:

The human player selects rock, paper, or scissors

The computer makes a random choice

Scores are tracked across multiple rounds

The game runs for a fixed number of rounds or until the player quits

##The project focuses on practicing:

Functions

Loops

Conditionals

User input handling

Basic game logic

##How to Play

Run the script in a browser environment (or console with prompt support)

Enter:

1 for Rock

2 for Paper

3 for Scissors

0 to quit

Check the console for round results and scores

Technologies Used

JavaScript (ES6)

Browser console

Learning Goals

Understand control flow with loops and conditionals

Practice function scope and variable management

Build simple game logic from scratch



Refactor logic for readability
